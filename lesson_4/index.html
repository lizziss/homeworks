CSS-властивості шрифтів

font-family

Визначає шрифт для тексту. Список шрифтів задається через кому: перший доступний шрифт використовується.

Типи шрифтів:

  • serif — шрифт з зарубками (засічками);
  • sans-serif — рублений (без зарубок);
  • fantasy — декоративні;
  • cursive — курсивні шрифти;
  • monospace — моноширинні.

font-size

Визначає розмір шрифту. Можна використовувати як абсолютні, так і відносні одиниці виміру.

Одиниці виміру:

  • px — пікселі;
  • em — відносні одиниці;
  • rem — відносні одиниці;
  • % — відсотки.

font-style

Визначає стиль шрифту. Можливі значення:

Можливі значення:

  • normal(за замовчуванням) - звичайний (не змінений, стандартний) стиль;
  • italic — курсив;
  • oblique — нахилений стиль.

font-weight

Визначає товщину шрифту.

Можливі значення(Числове значення - словесне значення):

  • 100 - Thin(Hairline)
  • 200 - Extra Light(Ultra Light)
  • 300 - Light
  • 400 - Normal(Regular)
  • 500 - Medium
  • 600 - Semi Bold(Demi Bold)
  • 700 - Bold
  • 800 - Extra Bold(Ultra Bold)
  • 900 - Black(Heavy)
  • 950 - Extra Black(Ultra Black)

CSS-властивості тексту

color

Встановлює текстовий колір.

Призначення кольорів у CSS:

  • Ключові слова для кольорів,такі як red , blue, green, black і т. д.
  • Шістнадцятковий формат (HEX) — шістнадцяткові значення, наприклад #FF0000 для червоного.
  • Формат RGB і RGBA — задає колір через значення червоного, зеленого і синього каналів у діапазоні від 0 до 255. RGBA додає прозорість через четвертий параметр (альфа-канал).
  • Формат HSL і HSLA — задає колір через значення відтінку (Hue), насиченості (Saturation) і яскравості (Lightness). HSLA додає прозорість через четвертий параметр (альфа-канал).

text-align

Вирівнює текст всередині блочного елемента

Значення властивості:

  • center - вирівнює по центру;
  • left - вирівнює по лівому краю ВСТАНОВЛЕНО ЗА ЗАМОВЧУВАННЯМ тобто не потрібно писати, тому що текст за замовчуванням вирівняно по лівій стороні;
  • right - вирівнює по правому краю;
  • justify - вирівнює текст по ширині об'єкта в якому він знаходиться, автоматично змінюючи пробіли між словами.

text-decoration

Додає до тексту підкреслення, накреслення зверху або перекреслення

Базові значення властивості^

  • line-through - закреслює текст;
  • overline - лінія над текстом;
  • none - (за замовченням) скасовує встановлений раніше параметр.
  • underline - лінія під текстом;

Додаткові властивості::

  • text-decoration-thickness - встановлює товщину лінії
  • text-underline-offset - встановлює відстань від тексту до лінії

text-shadow

Додає тінь до тексту. горизонтальне_зміщення вертикальне_зміщення розмір колір; Також може бути мультитінь, тіні записують через кому

Gриклад запису мультитіні з двох тіней:

text-shadow: 2px 2px 5px #000, 5px -2px 10px red;

text-transform

Змінює регістр (регистр) тексту.

Значення властивості:

  • capitalize - кожне слово починається з великої літери;
  • lowercase - усі літери прописні (нижній регістр);
  • uppercase - усі літери великі (верхній регістр);
  • none - (за замовчанням) не змінює (або скасовує раніше встановлене) значення.

line-height

Визначає міжрядковий інтервал (висота рядка) Розрахунок - line-height / font-size

Значення властивості:

  • Числове значення (1.5): відносна висота рядка до розміру шрифту.
  • Відсотки (150%): висота рядка у відсотках від розміру шрифту.
  • Фіксовані одиниці (наприклад, px): абсолютна висота рядка.

  • normal - (за замовчанням) не змінює (або скасовує раніше встановлене) значення.
